# 独家题库 更新日志

## v1.2 - 2026-07-03

### 新增功能
- **只看未做模式**：刷题页顶部新增「查看全部 / 只看未做」切换标签
  - 只展示未作答的题目
  - 隐藏「上一题」按钮，仅保留「下一题」
  - 当前题未答时「下一题」按钮置灰，作答后恢复可点
  - 作答后停留当前题，完整展示解析、知识点回顾、举一反三
- **防盗版保护**：禁止页面文字选中和右键菜单
- **答题音效**：答对 / 答错分别播放不同提示音，举一反三变式题同样生效
- **全部重置二次确认**：点击「全部重置」后弹出确认弹窗，明确提示会清空所有答题记录、恢复为未做状态

### 优化调整
- 「只看未做」模式进度条改为「未做题数（未做）/ 总题数」，进度条随做题从满到空
- 题号统一显示题目在全部题库中的固定序号
- 举一反三答题函数改为按题目 ID 定位，避免过滤模式下取错题

### 修复问题
- 修复「只看未做」模式下答题后自动跳转到下一题的问题
- 修复进入下一题后「下一题」按钮状态不重置的问题
- 修复举一反三题目点击选项后解析与题干不匹配的问题
